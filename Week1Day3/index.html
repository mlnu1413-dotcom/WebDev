Quick checklist (adoption-ready home)

  1. Choose a safe space for the first few days (quiet room or crate area).
  2. Buy basic supplies: food, bowl, leash, collar, ID tag, litter box (for cats).
  3. Schedule a vet visit within the first 7–14 days.
  4. Plan daily routine: meals, walks, playtime, and training.
